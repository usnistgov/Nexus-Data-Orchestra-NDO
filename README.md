## Project Purpose

Nexus Data Orchestra organizes repositories and resources used to support scientific data orchestration across related systems, including:

- **NexusLIMS**
  - CDCS-based frontend application for viewing, querying, and downloading NexusLIMS experimental XML records.
  - Docker deployment configuration for running the NexusLIMS-CDCS frontend and supporting services.
  - Backend system for generating experimental records from microscopy data, reservation metadata, and extracted file metadata.

- **NIST_Sample**
  - Repository for NIST sample tracking and sample-related data management.

- **Instrument DB**
  - Repository for NIST Instrument tracking and data management.
